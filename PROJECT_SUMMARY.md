## 🎨 Customization Examples

### Change Website Title
In `index.html` line 5:
```html
<title>My Awesome Thrift Store</title>
```

### Change Primary Color
In `css/styles.css` line 12:
```css
--primary-color: #your-color-here;
```

### Add New Product
In `index.html`, duplicate a product card:
```html
<div class="product-card" data-category="category-name">
    <div class="product-image"></div>
    <h3>Product Name</h3>
    <p class="price">$XX</p>
    <p class="condition">Condition</p>
    <button class="btn btn-small">Quick View</button>
</div>
```

### Update Contact Info
In `index.html`, contact section:
```html
<p><strong>Email:</strong> your-email@example.tn</p>
<p><strong>Phone:</strong> +216 XX XXX XXXX</p>
<p><strong>Location:</strong> Your Address</p>
```

## 🐛 Troubleshooting

### Site Not Loading?
1. Check file paths in HTML
2. Ensure all files are in correct folders
3. Check browser console for errors (F12)

### Styles Not Applying?
1. Hard refresh browser (Ctrl+F5)
2. Check CSS file path in HTML
3. Verify CSS file exists

### JavaScript Not Working?
1. Check browser console for errors
2. Verify script.js path in HTML
3. Ensure JavaScript is enabled

### Mobile Menu Not Working?
1. Check if JavaScript is enabled
2. Verify hamburger function exists
3. Check mobile breakpoint in CSS
